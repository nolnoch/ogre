Detailed Description

Networking wrapper for SDL_net created for OGRE engine use in CS 354R at the University of Texas at Austin, taught by Don Fussell in Spring 2014. This wrapper requires SDL_Net v1.2.8 for retrieving the user's local IP address. Compile as a static library or remove that functionality if the required version is not present on your system.
